# project-spec

This folder contains architectural decisions (ADRs) and project-level specifications for Trispr Flow.

## Structure

```
project-spec/
└── decisions/
    └── YYYY-MM-DD/
        └── slug.md   ← ADR files
```

ADRs are created when a decision is:
1. Hard to reverse
2. Surprising without context
3. The result of a real trade-off between genuine alternatives

For implemented decisions without ADR files, see `docs/DECISIONS.md`.
